# Security

This is a personal hobby project (see `README.md`) — no SLA, no guaranteed response time.

**Supported versions:** only the latest [release](https://github.com/yoramvandevelde/pitchforklauncher/releases).

**Reporting a vulnerability:** please use GitHub's private
["Report a vulnerability"](https://github.com/yoramvandevelde/pitchforklauncher/security/advisories/new)
instead of a public issue, so any real vulnerability isn't disclosed before there's a fix.
